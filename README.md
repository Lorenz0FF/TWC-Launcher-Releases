# The Walking Craft Launcher — Releases

Ce repo contient uniquement les binaires de release du launcher **The Walking Craft** (Windows / macOS / Linux). Le code source est privé.

## Téléchargement

Va sur la page **[Releases](https://github.com/Lorenz0FF/TWC-Launcher-Releases/releases/latest)** et choisis le fichier correspondant à ton OS :

| OS | Fichier |
|---|---|
| Windows | `TWC-Launcher-Setup-X.Y.Z.exe` |
| macOS (Intel + Apple Silicon) | `TWC-Launcher-X.Y.Z.dmg` |
| Linux | `TWC-Launcher-X.Y.Z.AppImage` |

## Auto-update

Une fois le launcher installé, il se met à jour automatiquement à chaque nouvelle release publiée ici. Pas besoin de revenir télécharger manuellement.
